# різним форматування

## через "##" позначають підзаголовки, а заголовки через "#". Рівень заголовка залежить від кількості "#"

## задопомогою "*" та "-", "+' позначають списки
* 1
* 2

- 1
- 2

+ 1
+ 2

## Цитати та змінна шрифту
**Жирний**
*Курсив*
>Цитата
